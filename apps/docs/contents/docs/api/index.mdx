A API do imobiliary oferece acesso a todas as funcionalidades
presentes dentro da plataforma, possibilitando acesso a API
utilizando-se desta documentação.

### Indice

#### Autenticação
- [Sobre a autenticação](./api/autenticacao)

#### Rotas
- [Rotas](./api/rotas)
  - [Usuário](./api/rotas/usuario)
    - `GET /users` - [Listar Usuários](./api/rotas/usuario/listar)
    - `GET /users/:user_id` - [Buscar Usuário](./api/rotas/usuario/buscar)
    - `POST /users` - [Criar Usuário](./api/rotas/usuario/criar)
    - `POST /users/auth` - [Autenticar Usuário](./api/rotas/usuario/autenticar)
    - `PUT /users/avatar` - [Alterar Avatar](./api/rotas/usuario/alterar-avatar)
  - [Imóvel](./api/rotas/imovel)
    - `GET /properties` - [Listar Imóveis](./api/rotas/imovel/listar)
    - `GET /properties/:property_id` - [Buscar Imóvel](./api/rotas/imovel/buscar)
    - `POST /properties` - [Criar Imóvel](./api/rotas/imovel/criar)
  - [Proprietário](./api/rotas/proprietario)
    - `GET /owners` - [Listar Proprietários](./api/rotas/proprietario/buscar)
    - `GET /owners/:owner_id` - [Buscar Proprietário](./api/rotas/proprietario/listar)
    - `POST /owners` - [Criar Proprietário](./api/rotas/proprietario/criar)
    - `PUT /owners/assign` - [Atribuir Proprietário](./api/rotas/proprietario/atribuir)
