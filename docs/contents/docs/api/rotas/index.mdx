### Rotas de Usuário
 - `GET /users` - [Listar Usuários](./rotas/usuario/listar)
 - `GET /users/:user_id` - [Buscar Usuário](./rotas/usuario/buscar)
 - `POST /users` - [Criar Usuário](./rotas/usuario/criar)
 - `POST /users/auth` - [Autenticar Usuário](./rotas/usuario/autenticar)
 - `PUT /users/avatar` - [Alterar Avatar](./rotas/usuario/alterar-avatar)
