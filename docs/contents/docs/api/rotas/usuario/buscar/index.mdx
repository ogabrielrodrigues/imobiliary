---
title: Buscar Usuário
description: Nessa rota você pode buscar um usuário através do id.
---

```http
GET https://api.imobiliary.com/users/:user_id
```

### Requisição
Para realizar essa requisição, deve-se informar um `id` de usuário válido
e assim trazer seus dados. Exemplo:
```http
GET https://api.imobiliary.com/users/0fabb50f-d83b-4c77-a2c9-c62bfecfac1c
```

### Respostas
Esses são as possíveis respostas desta requisição.

#### 200
Receberá essa resposta caso informe o `id` de um usuário existente.

```json
{
  "id": "0fabb50f-d83b-4c77-a2c9-c62bfecfac1c",
  "creci_id": "00000-F",
  "fullname": "Fulano de Tal",
  "cellphone": "(01) 12345-6789",
  "email": "fulanodetal@email.com",
  "avatar": "https://..." // caso tenha
}
```

#### 400
Receberá essa resposta caso informe um `id` de usúario inválido.

```json
{
  "code": 400,
  "message": "informe um id de usuário válido",
  "error": "Bad Request"
}
```

#### 404
Receberá essa resposta caso informe um `id` de um usuário que não existe.

<Note title="Obs" type="warning">
  Caso não informe o <code>id</code> de um usuário, será devolvida uma
  resposta sem o formato padrão de erro.
</Note>

```json
{
  "code": 404,
  "message": "usuário não encontrado ou não existente",
  "error": "Not Found"
}
```

#### 500
Receberá essa resposta caso ocorra um erro interno no servidor.
Isso pode acontecer por diversos motivos, como problemas de conexão com o servidor.

Caso o problema persista [Entre em contato](https://docs.imobiliary.com/contato)
```json
{
  "code": 500,
  "message": "erro interno do servidor",
  "error": "Internal Server Error"
}
