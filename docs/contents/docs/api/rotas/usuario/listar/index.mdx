---
title: Listar Usuários
description: Nessa rota você pode listar os usuários cadastrados na plataforma.
---

```http
GET https://api.imobiliary.com/users
```

### Requisição
Para realizar essa requisição, basta enviar requisição para a rota referente.

### Respostas
Esses são as possíveis respostas desta requisição.

#### 200
Receberá essa resposta caso não ocorra nenhum erro durante o processamento da
requisição. Essa resposta pode ser recebida com os registros dos usuários, ou
em caso de não haver nenhum usuário cadastrado receberá uma `Array` vazio como resposta.

Exemplo de resposta com usuários cadastrados:

```json
[
  {
    "id": "0fabb50f-d83b-4c77-a2c9-c62bfecfac1c",
    "creci_id": "00000-F",
    "fullname": "Fulano de Tal",
    "cellphone": "(01) 12345-6789",
    "email": "fulanodetal@email.com",
    "avatar": "https://..." // caso tenha
  }
]
```

#### 500
Receberá essa resposta caso ocorra um erro interno no servidor.
Isso pode acontecer por diversos motivos, como problemas de conexão com o servidor.

Caso o problema persista [Entre em contato](https://docs.imobiliary.com/contato)
```json
{
  "code": 500,
  "message": "erro interno do servidor",
  "error": "Internal Server Error"
}
