---
title: Logar Usuário
description: POST /users/auth
---

Nessa rota você pode logar um usuário no imobiliary.

Corpo da requisição:
```json
{
  "email": "fulanodetal@email.com",
  "password": "12345678"
}
```

Esses são os possíveis retornos nessa requisição:
<Tabs defaultValue="200" className="pb-1">
  <TabsList>
    <TabsTrigger value="200">200</TabsTrigger>
    <TabsTrigger value="400">400</TabsTrigger>
    <TabsTrigger value="401">401</TabsTrigger>
    <TabsTrigger value="404">404</TabsTrigger>
    <TabsTrigger value="500">500</TabsTrigger>
  </TabsList>
  <TabsContent value="200">
    Receberá o header `Authorization` essa resposta caso envie um corpo de requisição válido.

    ```http
    Authorization: Bearer <token>
    ```
  </TabsContent>

  <TabsContent value="400">
    Receberá essa resposta caso informe sem algum campo obrigatório ou um corpo de requisição inválido.

    Corpo da faltando o campo `password`:

    ```json
    {
      "email": "fulanodetal@email.com",
      "password": 95495495 // não é uma string
    }
    ```

    Resposta:

    ```json
    {
      "code": 400,
      "message": "Corpo de requisição inválido",
      "error": "Bad Request"
    }
    ```
    </TabsContent>

    <TabsContent value="401">
    Receberá essa resposta caso a senha não coincidir com a senha cadastrada.

    Corpo da requisição com o campo `senha` errado:

    ```json
    {
      "email": "fulanodetal@email.com",
      "password": "12345677" // senha errada
    }
    ```

    Resposta:

    ```json
    {
      "code": 401,
      "message": "A senha não coincide com a cadastrada",
      "error": "Unauthorized"
    }
    ```
    </TabsContent>

    <TabsContent value="404">
    Receberá essa resposta caso o email não coincidir com algum email cadastrado.

    Corpo da requisição com o campo `email` inesxistente:

    ```json
    {
      "email": "outrofulano@email.com",
      "password": "12345678"
    }
    ```

    Resposta:

    ```json
    {
      "code": 404,
      "message": "Usuário não encontrado ou não existe",
      "error": "Not Found"
    }
    ```
    </TabsContent>

    <TabsContent value="500">
    Receberá essa resposta caso ocorra um erro interno no servidor.
    Isso pode acontecer por diversos motivos, como problemas de conexão com o servidor.

    Caso o problema persista [Entre em contato](https://imobiliary.com/contato)

    ```json
    {
      "code": 500,
      "message": "Erro interno do servidor",
      "error": "Internal Server Error"
    }
    ```
    </TabsContent>
</Tabs>
